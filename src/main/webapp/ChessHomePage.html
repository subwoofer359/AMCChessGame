Introduction

The animation-fill-mode property specifies what styles will apply for the element when the animation is not playing (when it is finished, or when it has a "delay"). By default, CSS animations will not affect the element you are animating until the first keyframe is "played", and then stops affecting it once the last keyframe has completed. The animation-fill-mode property can override this behavior.
